# Cosmos Omni 2026 🌌

Portal astrológico para el rastreo de efemérides geocéntricas y ciclos lunares.

## Estructura del Proyecto
- `index.html`: Estructura principal y navegación.
- `styles.css`: Estilos visuales, animaciones espaciales y diseño responsive.
- `data.js`: Base de datos de posiciones planetarias y arquetipos.
- `script.js`: Lógica de renderizado dinámico y control de menú.

## Características
- Fondo animado dinámico.
- Menú de hamburguesa para dispositivos móviles.
- Renderizado automático de tablas lunares y efemérides.
