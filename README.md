# 🌌 Cosmos Omni 2026

**Cosmos Omni** es un portal astrológico de alta precisión diseñado para el seguimiento de tránsitos planetarios, ciclos lunares y el estudio de arquetipos mitológicos. Esta herramienta transforma datos técnicos en información visualmente digerible, permitiendo a los usuarios entender el "clima astral" de cada jornada.

---

## 🛠️ Tecnologías Utilizadas
Para el desarrollo de este proyecto se han utilizado estándares web modernos, priorizando la velocidad y la adaptabilidad (Responsive Design):

* **HTML5 Semantic:** Estructura organizada para una navegación tipo SPA (Single Page Application).
* **CSS3 Custom Properties:** Uso de variables para una estética estelar coherente y Media Queries para visualización en PC y Móviles.
* **JavaScript (Vanilla):** Lógica personalizada para el cálculo de fases lunares, renderizado de arquetipos y gestión de navegación dinámica.
* **Google Fonts:** Tipografías "Cinzel" y "Montserrat" para un acabado profesional y legible.

---

## 🚀 Funcionalidades Principales
1.  **Almanaque Lunar Pro:** Cálculo dinámico de grados, signos, elementos y dignidades de la Luna.
2.  **Biblioteca de Arquetipos:** Base de datos profunda con las deidades regentes de cada planeta.
3.  **Balance Elemental:** Gráficos que muestran la predominancia de Fuego, Tierra, Aire y Agua en tiempo real.
4.  **Efemérides Técnicas:** Control de retrogradaciones, sombras y estados planetarios.
5.  **Glosario Educativo:** Espacio dedicado a la enseñanza de conceptos astrológicos complejos.
